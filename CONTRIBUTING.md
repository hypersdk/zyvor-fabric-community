# Contributing

This is a public feedback repo — source code is not here.

## How to help

- Report bugs clearly with reproduction steps
- Request features with a real problem statement
- Vote on issues with 👍 to signal priority
- Answer questions in Discussions

## Labels

| Label | Meaning |
|-------|---------|
| `bug` | Something is broken |
| `enhancement` | New feature or improvement |
| `ux` | Layout, design, or usability issue |
| `docs` | Documentation gap |
| `question` | Needs clarification |
| `needs-triage` | Not yet reviewed |
| `priority-high` | Blocks common workflows |
| `priority-medium` | Important but has a workaround |
| `priority-low` | Nice to have |
